# PaaS-TA 컨테이너 플랫폼 KubeEdge 설치 
## 소개

컨테이너 플랫폼 Edge 배포용 쿠버네티스 설치를 위한 KubeEdge keadm 실행 파일 구성이다. keadm 명령을 통해 KubeEdge 설치 시 필요한 추가 파일을 자동으로 다운로드 한다.

## 스크립트 설명 
KubeEdge는 keadm 명령을 사용하기 위한 실행파일 1개로 구성 되어 있다.
