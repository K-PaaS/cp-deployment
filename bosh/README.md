# PaaS-TA 컨테이너 플랫폼 BOSH 배포 
## 소개
쿠버네티스 단독배포, Edge배포, CaaS배포를 사용하기 위한 Bosh 기반의 릴리즈의 설치 및 서비스를 등록 하는데 필요한 소스파일 구성이다.

## 스크립트 설명 
Components 요소 및 배포의 속성을 정의한 manifest와 IaaS마다 서버 환경에 맞춰 설정을 할 수 있는 deploy 파일들로 구성 되어있다.
